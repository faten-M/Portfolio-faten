# Portfolio-faten
Mini-projet Portfolio - Méthode BMAD

Ce projet est mini-portfolio réalisé dans le cadre de mon stage .Il présente mes projets , mes compétences ...

## Objectifs du projet:
- créer un portfolio simple et clair 
- Appliquer la methode BMAD
- Utiliser un workflow Git propre (branche , commit , pull request )
- Design reponsive 
- deployer  le site avec GitHub pages

## Méthode BMAD 
=> détailler  dans le fichier bmad.md

## Organisation Git :
chaque fonctionnalité est développée  dans une branche séparée :

- feature/structure
- feature/accueil (inclut la partie "À propos de moi")
- feature/projets
- feature/competences
- feature/contact
- feature/style
- feature/javascript

  
Chaque branche fait l’objet d’une **Pull Request**, avec un reviewer assigné.
